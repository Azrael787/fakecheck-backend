# FakeCheck Frontend

Минимальный скелет фронтенда. Здесь планируется интерфейс для загрузки файлов
и просмотра результата проверки.

## Идея интерфейса
- Поле загрузки файла (drag & drop).
- Выбор типа файла (фото/видео/другое).
- Кнопка "Проверить".
- Статус обработки и отображение процента фейка.

- Реализовать форму загрузки и опрос статуса.
