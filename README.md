# FakeCheck Backend

Backend-сервис для анализа изображений и оценки вероятности того, что изображение сгенерировано нейросетью.

Сервис принимает изображение, анализирует его по набору независимых признаков и возвращает:
- вердикт
- вероятность (в процентах)
- вклад каждого сигнала

---

## Что делает backend

Backend **сам принимает решение**, без ручной оценки:

- анализирует метаданные изображения
- проверяет шум и «идеальность» картинки
- анализирует края объектов
- проверяет корреляцию цветовых каналов
- объединяет всё в ансамбль
- возвращает итоговую вероятность

Результат **вероятностный**, а не бинарный.
